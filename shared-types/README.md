# Shared Types Package

Shared TypeScript types for SuCAR/SUKA Car Wash Booking System.

## Usage

### In Flutter (Dart)
Convert TypeScript types to Dart models using code generation or manual conversion.

### In Next.js
```typescript
import { Booking, User, BookingStatus } from '@sucar/shared-types';
```

## Building

```bash
npm install
npm run build
```

## Types Included

- User & UserRole
- Vehicle
- Service & ServiceName
- Booking & BookingStatus
- Payment & PaymentStatus
- CarWash
- Dashboard Stats
- API Responses
- Registration Data
